💬 Message Box with Character Counter
📌 Project Overview

This project is a Message Box web application built using HTML, CSS, and JavaScript.
The application allows users to type a message inside a text area while displaying a real-time character counter that updates as the user types.

The goal of this project was to practice DOM manipulation, event handling, and interactive UI behavior in JavaScript while also improving my front-end development skills.

🌐 Live Demo

🔗 https://chandrusk3456-tech.github.io/Message-box/

🚀 Features

Interactive message input box

Real-time character counter

Displays the number of characters typed

Shows remaining characters allowed

Dynamic updates while typing

Clean and simple user interface

Beginner-friendly front-end project

🛠️ Technologies Used

HTML5 – Structure of the message box

CSS3 – Styling and layout design

JavaScript – Real-time character counting functionality
